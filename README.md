# 🍃 StockSage | SME Decision Intelligence
**StockSage** is an intelligent inventory monitoring and decision support system designed specifically for Malaysian SMEs. Our platform addresses the "Decision Gap" in retail by transforming fragmented inventory data into actionable economic strategies, helping business owners reduce waste and maximize revenue.
### 📂 **Project Assets**
The required **System Analysis Documentation, Quality Assurance Testing Analysis Documentation, Product Requirement Documentation** and **Pitch Deck** have been attached within this repository for review. These documents outline our technical architecture, business logic, and market impact analysis.
### 📑 **Core Features**
 * **Inventory Health Dashboard:** Real-time visualization of current stock value and "Profit at Risk" due to nearing expiry dates.
 * **Intelligent Action Plan:** Automated generation of sales strategies (e.g., Quick-Clearance, Flash Sales) based on real-time stock status.
 * **Oracle DB Synchronization:** Persistent data management using Oracle Database 19c for secure and reliable inventory tracking.
 * **Live Stock Management:** Update inventory quantities and expiry dates directly from the dashboard with instant database persistence.
### 🛠️ **Technical Stack**
 * **Backend:** Java 17, Spring Boot v4.0.5.
 * **Database:** Oracle Database 19c.
 * **Frontend:** HTML5, CSS3 (Inter Font), and Bootstrap 5.
 * **Architecture:** Our system utilizes **ILMU GLM** as the core API architecture for managing strategic inference and decision-making logic.
### 🚀 **Installation & Setup**
 1. **Oracle Database:** Ensure your Oracle XE instance is active and the PRODUCTS table is created.
 2. **Configuration:** Update the database credentials in src/main/resources/application.properties.
 3. **Run Application:**
   * Navigate to the root folder.
   * Run ./mvnw spring-boot:run or execute StocksageApplication.java from your IDE.
 4. **Dashboard Access:** Open your browser and navigate to http://localhost:8081.
